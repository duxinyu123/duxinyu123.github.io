## 3. 要求
1. 必须是顺序存储结构
2. 必须按照关键字大小有序排列

## 4. 代码实现
```python
def binary_search(l, item):
    length = len(l)
    if length == 0:
        return False
    mid = length//2
    if item == l[mid]:
        return True
    elif item < l[mid]:
        return binary_search(l[0:mid], item)
    elif item > l[mid]:
        return binary_search(l[mid+1:], item)

print(binary_search([1,2,3,4,5,6,7], 7))
```
