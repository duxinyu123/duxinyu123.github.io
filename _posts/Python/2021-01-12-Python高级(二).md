# 一、property属性
## 1.作用：把方法的调用变成属性调用

```python
class Person():
    def __init__(self):
        self.__age = 0

    @property
    def age(self):
        return self.__age

    @age.setter
    def age(self,age):
        self.__age = age


p = Person()
p.age = 10
print(p.age)
```

## 2.类属性方式

```python
class Person():
    def __init__(self):
        self.__age = 0

    def get_age(self):
        print('这里获取了age')
        return self.__age

    def set_age(self,age):
        print('这里设置了age')
        self.__age = age

    # 属性中第一个是get方法 第二个是set方法
    
    age = property(get_age, set_age)


p = Person()
p.age = 10
print(p.age)
```
